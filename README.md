# texo
A simple build system based on bash and nixpkgs.

## requirements

- bash (duh)
- nixpkgs
- curl

## usage
1. edit texo source file and fill out the config section
2. create dev environment

```bash
$ ./texo init
```
3. enter dev evironment

```bash
$ ./texo sh
```

4. start coding & compile your project with:

```bash
$ ./texo com
```
